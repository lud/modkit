# Modkit

This library contains a small set of tool to work with Elixir modules files.


## Installation


The package can be installed by adding `modkit` to your list of dependencies in
`mix.exs`:

```elixir
def deps do
  [
    {:modkit, "~> 0.1.0"}
  ]
end
```
